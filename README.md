Project Overview
The AI Machine Learning Learning Assistant is a web-based application designed to help students understand Machine Learning concepts through practical interaction with datasets. The system combines dataset analysis, algorithm recommendation, model accuracy comparison, and a chatbot assistant into a single learning platform.
The goal of this project is to make Machine Learning learning interactive, beginner-friendly, and practical.
Features
The application includes the following modules:
1. User Login Module
Simple login interface
Session management using local storage
Dashboard access after login
2. Dataset Analysis Module
Upload CSV dataset
Dataset preview
Missing value detection
Column type identification
Dataset summary
3. Algorithm Recommendation Module
Target column selection
Problem type detection (classification or regression)
Suitable algorithm suggestions
4. Accuracy Analysis Module
Model training
Accuracy comparison
Best model identification
Accuracy graph visualization
5. Chatbot Assistant Module
Machine Learning concept explanation
Student learning support
Integrated using Pickaxe AI
